# DSA

## GUI demo (Human-Robot Interaction)

Run the MATLAB GUI with:

```matlab
run('demos/demo_hri_gui.m')
```

The GUI includes:
- 10 customizable query types for place lookup and navigation-style requests.
- Linked-list baseline (linear scan) and KD-tree nearest lookup comparison.
- Timestamped speculative dialogue simulation with incremental replanning.
